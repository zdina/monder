ons-splitter

sliding-menuとsplit-view両方の機能を持つレイアウトです。

  • 使い方
  • 属性
  • メソッド
  • 関連情報
  • 議論

使い方 # 

<ons-splitter>
  <ons-splitter-content>
    ...
  </ons-splitter-content>

  <ons-splitter-side side="left" width="80%" collapse>
    ...
  </ons-splitter-side>
</ons-splitter>

関連情報 #

  • JavaScriptからコンポーネントを呼び出す

属性 #

名前 型 / デフォルト値 概要
var String このスプリットビューコンポーネントを参照するための名前を指定します。 Optional 初期化時のみ有効 angular
ons-destroy Expression "destroy"イベントが発火された時の挙動を独自に指定できます。 Optional 初期化時のみ有効 angular

closeRight([options]) 右のcollapseモードになっているons-splitter-side要素を閉じます。
closeLeft([options]) 左のcollapseモードになっているons-splitter-side要素を閉じます。

loadContentPage(pageUrl) ons-splitter-content用紙に表示されるページをpageUrlに指定します。

rightIsOpened() 右のons-splitter-side要素が開いているかどうかを返します。
getDeviceBackButtonHandler() ons-splitter要素に紐付いているバックボタンハンドラを取得します。
on(eventName, listener) イベントリスナーを追加します。 angular
once(eventName, listener) 一度だけ呼び出されるイベントリスナーを追加します。 angular
off(eventName, [listener]) イベントリスナーを削除します。もしイベントリスナーを指定しなかった場合には、そのイベントに紐づく全てのイベントリスナーが削除されます。 angular
