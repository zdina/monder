ons-progress

マテリアルデザインのprgoressコンポーネントです。linearもしくはcircularなプログレスインジケータを表示できます。

  • 使い方
  • 属性
  • 議論

使い方 # 

<ons-progress
 type="circular"
 value="55"
 secondary-value="87">
</ons-progress>

属性 #

名前 型 / デフォルト値 概要
type String indicatorのタイプを指定します。"bar"もしくは"circular"を指定できます。デフォルトは"bar"です。 Optional 初期化時のみ有効
modifier String プログレスインジケータの見た目を変更します。 Optional
value Number 現在の進行状況の値を指定します。0から100の間の値を指定して下さい。 Optional
secondary-value Number 現在の２番目の進行状況の値を指定します。0から100の間の値を指定して下さい。 Optional
indeterminate この属性が設定された場合、ループするアニメーションが表示されます。 Optional
