ons-progress

A material design progress component. Can be displayed both as a linear or circular progress indicator.

  • Usage
  • Attributes
  • Discussion

Usage # 

<ons-progress
 type="circular"
 value="55"
 secondary-value="87">
</ons-progress>

Attributes #

Name Type
Default Value		 Description
type String The type of indicator. Can be one of either "bar" or "circular". Defaults to "bar". Optional Works only during initialization
modifier String Change the appearance of the progress indicator. Optional
value Number Current progress. Should be a value between 0 and 100. Optional
secondary-value Number Current secondary progress. Should be a value between 0 and 100. Optional
indeterminate If this attribute is set, an infinite looping animation will be shown. Optional
