Attributes #

Name Type
Default Value		 Description
label String Text for animated floating label. Optional
no-float If this attribute is present, the label will not be animated. Optional
ng-model Bind the value to a model. Works just like for normal input elements. Optional
ng-change Executes an expression when the value changes. Works just like for normal input elements. Optional
