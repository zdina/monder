ons-ripple

Adds a Material Design "ripple" effect to an element.

  • Usage
  • Attributes
  • Discussion

Usage # 

<ons-list>
  <ons-list-item>
   <ons-ripple color="rgba(0, 0, 0, 0.3)"></ons-ripple>
   Click me!
  </ons-list-item>
</ons-list>

<ons-ripple target="children" color="rgba(0, 0, 0, 0.3)">
  <p>Click me!</p>
</ons-ripple>

Attributes #

Name Type
Default Value		 Description
color String Color of the ripple effect. Optional
center If this attribute is set, the effect will originate from the center. Optional
target String If this attribute is set to children, the effect will be applied to the children of the component instead of the parent. Optional
disabled If this attribute is set, the ripple effect will be disabled. Optional
